# SignalCheck

SignalCheckは、株式投資に関心を持ち始めた初心者が、投資判断の前に「自分がどのような情報を根拠にしているのか」を整理するためのリスク可視化ツールです。

このプロジェクトは、特定の銘柄について「買うべき」「売るべき」と判断するものではありません。投資判断に使っている情報源の偏りや、確認できていない観点に気づくための補助ツールです。

## 背景

NISAやネット証券、投資系SNS、YouTubeなどの広がりによって、学生や若年層でも株式投資に触れやすくなっています。

一方で、初心者はSNSの投稿、動画、ニュース、友人の話、株価の短期的な動きなど、多くの情報に囲まれています。情報が多いこと自体は便利ですが、その情報がどれくらい信頼できるのか、自分が何を根拠に判断しているのかを整理するのは簡単ではありません。

SignalCheckは、この「情報が多すぎる中で、自分の判断根拠を見失いやすい」という課題に注目しています。

## このアプリでできること

- 企業名や銘柄名を入力する
- 投資を考えた理由を入力する
- 参考にした情報源を選択する
- 情報源の偏りを確認する
- 話題性依存、感情依存、確認不足などのリスクを表示する
- 追加で確認すべき項目を提示する

## このアプリでやらないこと

- 銘柄の推奨
- 買い時や売り時の判断
- 株価予測
- 企業価値の評価
- 金融アドバイス
- 利用者自身の調査や判断の代替
