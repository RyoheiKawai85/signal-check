# 課題定義

## 社会背景

NISA、ネット証券、スマートフォンアプリ、SNS、YouTubeなどの普及により、個人が株式投資に触れるハードルは下がっています。

これにより、初心者が投資について学び始める機会は増えました。一方で、利用者は信頼性や目的が異なる多くの情報源に囲まれるようになっています。

## 中心となる課題

投資初心者は、多くの情報にアクセスできる一方で、その情報の質、偏り、限界をどのように評価すればよいか分からないことがあります。

その結果、次のような理由で判断してしまう可能性があります。

- SNSで話題になっている
- 株価が短期的に上がっている
- 友人や知人にすすめられた
- 有名企業なので安心に感じる
- 配当利回りが高く見える
- ひとつのニュース記事だけを見た
- ランキングやまとめ投稿を見た

問題は、単に「良くない銘柄を選ぶかもしれない」ということだけではありません。より重要なのは、なぜ自分がその判断を妥当だと感じているのかを理解しないまま意思決定してしまうことです。

## 利用者の困りごと

- 「SNSで多くの人が話しているから安心に感じる」
- 「株価が上がっているから良い企業に見える」
- 「配当利回りが高いから魅力的に見える」
- 「知っている企業名だから安心してしまう」
- 「他に何を確認すればいいのか分からない」

## 仮説

初心者が、投資を考える理由や参考にした情報源を整理できれば、自分の判断がどこに偏っているのか、どの確認が不足しているのかに気づきやすくなるのではないか。

SignalCheckは、この振り返りのプロセスを支援することを目指します。

## 対象範囲

SignalCheckは、投資判断の前段階にある「情報整理」と「リスクへの気づき」を対象にします。

やらないこと:

- 買うべき、売るべきという判断
- 株価予測
- 企業価値の算定
- 金融アドバイス

やること:

- 情報源を整理する
- 偏りの可能性を示す
- 確認不足の観点を示す
- 追加調査を促す

## プロジェクトの価値

このプロジェクトの価値は、投資を単なるお金の判断ではなく、情報をどう扱い、どう意思決定するかという問題として捉える点にあります。

関連するテーマ:

- 金融リテラシー
- 情報の信頼性
- リスク認識
- 利用者視点のシステム設計
- デジタルサービスの責任ある利用
